    • User Testing

    We tested our users by giving them a set of tasks to do on the website to test out how smoothly the navigation is and the overall user experience. The following tasks were:

    • Find information about the different gym possibilities

    • Find the phone number to StudentCompass

    • Find information about Ladestien

    • Find information about Lerkendal Studentby

    • What was the overall experience?

    Tester Audience

    The test subjects were all a part of our target group which were students.

    Testing Platforms

    • Browsers used: Chrome

    • Devices used: Macbook, iPad, Lenovo Thinkpad

    Feedback Summary

    Test 1: Chrome on Lenovo Thinkpad

    1. When asked about finding information about the different gym possibilities, the user immediately navigated to the “Activities”-option in the nav-bar. The user then assumed the goal was to locate “Walks”. After exploring the webpage and checking out the different tabs the user eventually located the information.

    2. The user quickly located the phone number and found it intuitive that it was located in the footer, especially considering that the website does not have an “About us” page that provides more information.

    3. The user found the information of both Ladestien and Lerkendal studentby easily.

    4. The website provided the user with an overall positive experience. The user complimented the design and overall good usability in regard to both font size and organization.

    Test 2 - Chrome on Macbook:

    1. I expected to find information about gym-possibilities under “activities”. I think the tab “SIT offerings” should have a different name, it’s not very intuitive which information you can find there.

    2. Suggestion for changes: Keep “activities” as a drop-down menu and have “SIT Gym” and “SIT Housing” as independent elements in the nav bar.

    3. The user commented that the rest of the navigation was good and the website was visually aesthetically pleasing

    Test 3 - Chrome on MacBook:

    1. It’s not very intuitive to understand which information you find under the different tabs in the nav bar.

    2. The UI is very nice

    Improvements from feedback

    Improvements for feedback from test 1:

    The navbar needs to provide the user with more information. A solution can be to add more options or change the names of the different tabs to make it more clear.

    Improvements for feedback from test 2 and 3:

    To make the navbar more intuitive, we can remove the drop-down meny “SIT offerings” and make them individual elements in the navbar.
